## 概要
- シークバー(音楽の再生、停止、スキップなど)を制御するためのキーボード
- 5スイッチ、1ロータリーエンコーダ(ボタン付き)
- 分割キーボード対応
- LED対応
- 完全オープンで、各自発注可能

## デフォルトキーマップ
```
┌───┬───┬───┐
│ A │ B │ C │
├───┼───┼───┤
│ D │ E │ F │ // Dはロータリーエンコーダ
└───┴───┴───┘
```

| key | single tap         | double tap     | other action   |
| --- | ------------------ | -------------- | -------------- |
| A   | 少し戻る           | 前のトラックへ |                |
| B   | 一時停止/再生      |                |                |
| C   | 少し進む           | 次のトラックへ |                |
| D   |                    |                | 回転で音量調節 |
| E   | ファンクションキー |                |                |
| F   | デバイスをスリープ |                |                |
